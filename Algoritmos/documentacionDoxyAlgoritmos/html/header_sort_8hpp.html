void insertionSort ( int  arr[],
int  n 
)

Ordena un array de enteros utilizando el algoritmo Insertion Sort.

Insertion Sort es un algoritmo de ordenamiento que recorre el array y en cada paso inserta el elemento actual en su posición correcta en la porción ordenada del array que está a su izquierda, repitiendo este proceso hasta que todo el array esté ordenado.

Parámetros
arrEl array de enteros a ordenar.
nEl número de elementos en el array.

Mueve los elementos mayores que key hacia adelante para insertar key en la posición correcta

◆ quickSort()

void quickSort ( int  arr[],
int  low,
int  high 
)

Ordena un array de enteros utilizando el algoritmo QuickSort.

El algoritmo QuickSort es eficiente y divide el array en dos sub-arrays, los cuales se ordenan de forma recursiva. Se utiliza un elemento pivote para reorganizar los elementos del array, de forma que los elementos menores al pivote estén a la izquierda y los elementos mayores estén a la derecha.

Parámetros
arrEl array de enteros a ordenar.
lowEl índice del primer elemento en el array.
highEl índice del último elemento en el array.
